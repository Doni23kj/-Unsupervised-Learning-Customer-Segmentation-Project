Customer Segmentation Project

Описание проекта

Данный проект представляет собой веб-приложение для сегментации клиентов с использованием алгоритмов машинного обучения. Система анализирует данные клиентов и автоматически делит их на группы (кластеры) в зависимости от их поведения.

Сегментация выполняется на основе двух ключевых параметров: баланса клиента и суммы его покупок.

Цель проекта

Целью проекта является разработка простой системы, позволяющей анализировать клиентов и определять их тип (сегмент), что может быть полезно для бизнеса, маркетинга и анализа поведения пользователей.

Установка и запуск

Клонировать репозиторий:
git clone https://github.com/Doni23kj/-Unsupervised-Learning-Customer-Segmentation-Project.git
cd customer_segmentation_project

Создать виртуальное окружение:
python -m venv .venv
source .venv/bin/activate

Установить зависимости:
pip install flask pandas numpy scikit-learn joblib

Запустить обучение модели:
python train_model.py

Запустить приложение:
python app.py

Открыть в браузере:
http://127.0.0.1:5000

Использование

Пользователь вводит следующие параметры:

Баланс клиента (BALANCE)  
Сумма покупок (PURCHASES)  

После ввода данных система определяет, к какому кластеру относится клиент, и выводит результат.

Клиенты делятся на следующие категории:

Пассивный клиент  
Стабильный клиент  
Активный клиент  
Перспективный клиент  
VIP клиент  

Примечание

Модель обучается заранее и не создаёт новые кластеры при каждом вводе данных. Она лишь определяет наиболее подходящую группу для нового клиента.
